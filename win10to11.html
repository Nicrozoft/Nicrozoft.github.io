Windows 10 仿 Windows 11

Windows 10 仿 Windows 11。
系统更改：
1.替换了所有壁纸和主题。
2.更改了系统图标。
3.将系统样式替换为 Windows 11 的。
4.设置了圆角窗口。
5.替换了所有系统音效。
6.添加了 Windows 11 启动音乐。
7.为 Microsoft Edge 设置圆角标签页。
8.替换了启动图标。
9.安装了 Start 11，修改了开始菜单样式，模仿 Windows 11 样式。
10.安装了 WindowFX，将窗口动画替换为 Windows 11 样式。
11.禁用了自动更新。
12.更新用户头像。
13.调整了部分设置。
14.调整了部分样式。
15.安装了 Windows 11 的新版应用并安装中文语言，包括：记事本、画图、截图工具、便笺。
