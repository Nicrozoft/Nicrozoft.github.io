Windows 7 仿 Windows 11

Windows 7 仿 Windows 11。
系统更改：
1.更改主题，壁纸，内置 Windows 11 的 6 个主题，将“中国”主题的壁纸替换为 Beauty of China PREMIUM 主题的 4K 壁纸（主题来源：Microsoft Store）
2.更改系统图标
3.更改版本号
4.更改声音
5.更改启动动画
6.将任务管理器替换为 DBC Task Manager x64
7.安装 Microsoft Edge 109 和 IE 11，并通过 Image File Execution Options 在 IE 打开时跳转到 Edge（如果确实要使用 IE，你可以删除注册表项 HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\iexplore.exe）
8.【2.0 内容】优化了主题设置，将发光和捕获的动作设为暗色主题，修复暗色主题的字体问题
9.【2.0 内容】添加了 4 个高对比度主题（1.0 版本直接移除了这些主题），并设置为 Windows 11 的高对比度主题样式
10.【2.0 内容】安装了 .NET Framework 4.5
11.【2.0 内容】安装了 QTTabBar 并汉化，设置了资源管理器、控制面板多标签页功能。
12.【2.0 内容】修复了 IE 跳转 Edge 时会自动下载文件的问题
13.【2.0 内容】将 Winver 等界面的图片替换为 Windows 11 原版。
14.【2.0 内容】安装 Sysinternals Desktops，支持虚拟桌面。
15.【3.0 内容】安装了 .NET Framework 4.8，以及所有版本的 VC++。
16.【3.0 内容】安装了 VxKex 扩展内核。
