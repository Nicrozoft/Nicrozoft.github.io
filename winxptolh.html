Windows XP 仿 Longhorn 优化版

Windows XP 仿 Windows Longhorn 优化版！
此产品基于原版 Windows XP SP3 修改。

系统更改：
1.替换主题以模仿 Longhorn Jade 样式。
2.替换了图标。
3.安装了 MiniBrowser 和 7-Zip。
4.安装了 Vista 界面仿真器，并调整了侧边栏。
5.添加了大量壁纸。
6.替换登录界面以仿照 Longhorn 样式。
7.替换了启动界面。
8.安装了微软雅黑字体并设为默认字体/
9。安装了部分更新。
