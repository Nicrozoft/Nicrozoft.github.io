Windows 10 优化版 v1.0

Windows 10 优化版 64位。
系统基于 Windows 10 x64 22H2 专业工作站版 最终版本。
此系统并不基于任何其他修改版。除安装程序外, 也没有参考其他任何修改版。只有安装程序使用了 Rectify 11 的部分文件。
此系统使用 Dism 封装，并使用 Wimlib-ImageX (1.0 使用 Dism++) 将镜像转为 ESD 格式并添加卷信息。我们坚决抵制任何流氓软件！抵制流氓封装工具 Easy Sysprep！
系统更改：
1.安装了所有版本的 Microsoft Visual C++ 运行库。 2.安装了 Office 2024 专业增强版并用 Ohook 模式永久激活。
3.安装了 Visual Studio Code。
4.安装了 Adobe Acrobat 2025、 Premiere 2025、Acrobat Pro 2025、After Effects 2025。
5.安装了新版微软雅黑 (11.3.0) 的 SemiBold、SemiLight、Heavy 三个字号。
6.内置了 80+ 款主题。
7.为 Edge 安装 Adguard 插件。
8.调整了任务栏的 Acrylic 透明度。
9.安装了 7-Zip、 Everything、 Files App， 以及火绒安全软件。
10.安装了 Windows Terminal 并设置为默认命令行软件。
11.内置了 VMware Tools 清理工具。
12.安装了 PowerToys。
