Windows 7 仿 Longhorn 优化版

Windows 7 仿 Longhorn #1 系统截图（2.0）



Windows 7 仿 Longhorn #2 系统截图（2.0）

该页面发布了两个 Windows 7 仿 Longhorn 系统。
注: 这两个系统是不同的系统，并不是同一个系统的不同版本，因此体验有所不同，有条件的建议分别体验一下。

#1 系统介绍及更改：
1.系统基于来源于网络的 LH_USA.6.0.4096.Lab06_N.230111-2145
2.安装了 Windows 7 中文语言包。
3.安装了 VMware Tools 12。
4.启用了 Aero 和 Longhorn Aurora。
5.安装了 Microsoft Edge。
6.翻译了一些系统文档。
7.【3.0 内容】微调了一些系统设置。
8.【3.0 内容】将开始菜单、WinAMP 等组件汉化。
注意: 在 VMware Workstation 虚拟机中安装该系统需要将硬件兼容性调整为 VMware 9.x 。

#2 系统介绍及更改：
1.系统基于原版 Windows 7 Service Pack 1。
2.安装了 Windows 7 仿 Windows Longhorn 美化包 v0.3，作者: John-PC。
3.添加了 Longhorn Aurora 程序并默认启用。
4.替换了 shellbrd。
5.安装了 Microsoft Edge。
6.安装了一些更新。
7.【3.0 内容】微调了一些系统设置。

在 3.0 版本开发过程中，提供帮助的人士: WinBetaUser
